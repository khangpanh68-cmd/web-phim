**Khang Panh68 Studio** là một ứng dụng web xem phim chuyên nghiệp được tích hợp hệ thống **Cloud Save**. Dữ liệu được đồng bộ thời gian thực, cho phép người dùng xem danh sách phim và ông trời có thể quản lý (thêm/xóa) phim trực tiếp trên đám mây.

## ✨ Tính năng nổi bật

- ☁️ **Cloud Sync:** Lưu trữ dữ liệu trên Google Firebase, không bị mất khi đổi thiết bị.
- ⚡ **Real-time Update:** Tự động cập nhật danh sách phim ngay lập tức khi có thay đổi từ phía Admin.
- 🔐 **quản lý Panel:** Hệ thống đăng nhập bảo mật dành riêng cho ông trời để quản lý nội dung.
- 📱 **Mobile Friendly:** Giao diện tối ưu hoàn hảo cho điện thoại (đặc biệt là khi chạy trên Acode).
- 🎬 **Video Player:** Tích hợp trình phát video YouTube mượt mà với hiệu ứng Modal.
- 🔎 **Search & Filter:** Tìm kiếm phim nhanh chóng và lọc theo thể loại (Hành động, Chiến tranh, AI).

## 🛠 Công nghệ sử dụng

- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6 Modules).
- **Backend:** Firebase Firestore (NoSQL Database).
- **Icons & Media:** YouTube API, Google Fonts.

## 🚀 Hướng dẫn cài đặt cho lập trình viên

1. **Tải mã nguồn:** Copy file `index.html` vào trình biên tập mã (Acode, VS Code...).
2. **Cấu hình Firebase:**
   - Tạo một dự án tại [Firebase Console](https://console.firebase.google.com/).
   - Kích hoạt **Cloud Firestore** ở chế độ thử nghiệm (Test Mode).
   - Đăng ký ứng dụng Web và lấy đoạn `firebaseConfig`.
3. **Kết nối:**
   - Dán đoạn mã `firebaseConfig` của bạn vào phần script trong file code.
4. **Chạy ứng dụng:** Mở file bằng trình duyệt và tận hưởng!

## 📸 Giao diện ứng dụng

- **Trang chủ:** Danh sách 19 phim gốc với hiệu ứng đổ bóng và bo góc hiện đại.
- **Thanh tìm kiếm:** Hiệu ứng focus màu đỏ đặc trưng của Khang Studio.
- **Admin Mode:** Bật lên khi đăng nhập đúng tài khoản và mật khẩu định sẵn.
