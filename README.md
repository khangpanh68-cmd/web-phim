## 🚀 Cách chạy local

1. Tải project về
2. Mở file `index.html`
3. Chạy trực tiếp trên trình duyệt

---

## 🌍 Deploy lên GitHub Pages

### Bước 1:
Tạo repository mới trên GitHub

### Bước 2:
Upload file `index.html`

### Bước 3:
Vào Settings → Pages:
- Branch: `main`
- Folder: `/root`

### Bước 4:
Truy cập link GitHub Pages
